Co znaczy skrót ID? ID to identyfikator – unikalna nazwa przypisywana pojedynczemu elementowi HTML.
Jaki jest cel stosowania ID? Służy do jednoznacznego wskazania danego elementu, np. w celu zastosowania stylów CSS lub wykonania akcji w JavaScript.
Jakie jest znaczenie znaku # W CSS symbol # jest używany do selekcji elementu na podstawie jego identyfikatora (ID).
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W CSS ID wywołuje się przez dodanie # przed nazwą identyfikatora, np. #naglowek { color: red; } – oznacza to, że element o ID „naglowek” będzie miał czerwony kolor tekstu
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia. 1. Każdy identyfikator musi być unikalny w ramach jednej strony HTML.
2. Nazwa ID powinna zaczynać się od litery i nie może zawierać spacji.
3. ID można używać nie tylko do stylizacji, ale też do tworzenia kotwic (linków wewnętrznych) oraz manipulowania elementami przez JavaScript.
